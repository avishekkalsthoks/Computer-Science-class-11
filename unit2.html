Boolean Algebra and Logic Gates

Boolean Algebra: Boolean algebra is the algebra of logic, dealing with binary variables (0 and 1). It is used to analyze and design logic circuits. It was introduced by George Boole, and is fundamental to the operation of modern digital computers.

Boolean Variable: A Boolean variable can only take one of two possible values: 0 (False) or 1 (True). These variables are typically denoted by letters such as A, B, X, Y, etc.

Boolean Function (Logic Function): A Boolean function is an expression formed by binary variables and operators (AND, OR, NOT) that produces an output (either 0 or 1) based on a given set of inputs. Example: F = XYZ’ + XY

Boolean Operators

AND Operator: The AND operator is represented by “.”. It produces a result of 1 only when both inputs are 1. Example: A AND B is represented as A.B.

OR Operator: The OR operator is represented by “+”. It produces a result of 0 only when both inputs are 0. Example: A OR B is represented as A+B.

NOT Operator: The NOT operator negates its operand. It is represented as A’. It produces the inverse of the input: 1 for 0 and 0 for 1.

Logic Gates

AND Gate: This gate produces a high output (1) only when both inputs are high. It is represented as Y = A.B.

OR Gate: This gate produces a high output (1) if at least one input is high. It is represented as Y = A+B.

NOT Gate: This gate inverts the input. It is represented as Y = A’ for input A.

NAND Gate: The NAND gate is the inverse of the AND gate. It produces 1 when at least one input is 0. It is represented as Y = (A.B)’.

NOR Gate: The NOR gate is the inverse of the OR gate. It produces 1 only when both inputs are 0. It is represented as Y = (A+B)’.

Exclusive OR (XOR) Gate: This gate produces a high output (1) when the inputs are different. It is represented as Y = A’.B + A.B’.

Exclusive NOR (XNOR) Gate: This gate produces a high output (1) when the inputs are the same. It is represented as Y = A.B + A’.B’.

De-Morgan's Theorem

First Theorem: The complement of a sum is the product of the complements. It is represented as (A+B)’ = A’.B’.

Second Theorem: The complement of a product is the sum of the complements. It is represented as (A.B)’ = A’+B’.
