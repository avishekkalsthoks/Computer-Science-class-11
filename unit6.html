Introduction to HTML

What is HTML?

HTML stands for Hyper Text Markup Language, which is used to create web pages for exchanging data and information over the internet. HTML files themselves are text files containing several tags and attributes. HTML was developed by Tim Berners-Lee and is based on SGML (Standard Generalized Markup Language). Web browsers interpret these tags and attributes to display the content of the page. The different versions of HTML include: HTML, HTML+, HTML 1.0, HTML 2.0, HTML 3.2, HTML 4.01, and now HTML 5.0.

Use of HTML

  • It is used to create the basic layout or design of web pages.
  • Without HTML, the World Wide Web wouldn't exist.
  • It allows embedding of images, videos, audio, etc.
  • It allows customization and formatting of characters such as bold, italic, underline, etc.
  • It helps in creating links and lists.

Advantages of HTML

  • It is easy to use, flexible, and user-friendly.
  • Supported by all web browsers and operating systems (Macintosh, Windows, Unix, etc.).
  • Content is easier to modify and obtain.
  • Compatible with all search engines.
  • HTML 5.0 comes with several new and advanced features.
  • HTML files take a short period of time to load.
  • Several tutorials are easily available for learning HTML.

Disadvantages of HTML

  • It is difficult to create attractive and interactive web pages using only HTML (other technologies like PHP, JavaScript, SQL, etc. are needed).
  • HTML cannot be used to create dynamic web pages.
  • HTML is a scripting language, not a programming language.
  • There are several incompatibilities with HTML versions.
  • No complete, universally accepted standard for HTML.

HTML Tags

HTML tags are the instructions or commands given to the web browser to organize text, images, and other web page components. HTML tags are written inside angular brackets <>.

Types of HTML Tags

  • Pair Tags: Tags that have both opening and closing tags. They are also called container tags. The opening tag is written with <tag> and the closing tag with </tag>.
  • Singular Tags: Tags that have only an opening tag with no closing tag. These are also called empty tags (e.g., <img>).

Note:

HTML tags and attributes are not case-sensitive.

Attributes in HTML

Attributes are the descriptive properties of HTML tags that provide additional features. They allow designers to customize and modify several components of web pages according to their requirements.

For example, the <font> tag has the following attributes:

  • bgcolor: Used to set the background color.
  • text: Used to change the color of the text.
  • background: Used to set an image as the background.

Steps to Write and Execute an HTML Document

  • Write all HTML tags, attributes, and related information in a text editor (e.g., Notepad).
  • Save your document with the .html extension (e.g., sample.html).
  • Execute the HTML file by opening it in any web browser.

Common Terminologies in Web Designing

Webpage

A webpage is an HTML document or a document in another scripting language that displays information in the form of text, images, or other components over the internet.

Website

A website is a collection of interrelated web pages connected by hyperlinks. It contains information and resources that may be related across different pages.

Home Page

The introductory page that loads first when a website is accessed.

Web Browser

A web browser is an application that accesses and displays web pages via the HTTP protocol. Examples: Google Chrome, Internet Explorer (Microsoft Edge), Safari, Mozilla Firefox, etc.

URL

URL stands for Uniform Resource Locator. It is the standard way of referencing a protocol, host, and directory. For example: https://readersnepal.com

DBMS (Database Management System)

Data vs Information

Data: Raw facts and figures that have no meaning until processed (e.g., Ram, 19, 12, Pokhara).

Information: Meaningful data processed into a useful form (e.g., Ram is a 19-year-old student from Pokhara-12).

What is DBMS?

A DBMS (Database Management System) is a software system used to manage and organize data. It helps store, process, and retrieve data efficiently. Examples: MS-Access, MySQL, Oracle, FoxPro, dBase.

Advantages/Features/Importance of DBMS

  • Quick and easy data searching.
  • Reduces data redundancy (repetition).
  • Allows data sharing over a network.
  • Improves data security and privacy.
  • Improves data consistency (reduces variance).
  • Provides an organized way of storing data.

Disadvantages of DBMS

  • Installation and operating costs are higher.
  • Requires more disk space.
  • May have security and privacy concerns.
  • Requires technical manpower (DBA) for maintenance and operation.
