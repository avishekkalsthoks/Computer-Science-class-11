C Programming Language

What is C Programming Language?

C is a High-Level Programming Language developed by Dennis Ritchie at AT&Tâ€™s Bell Laboratories in the USA in 1970. It is a powerful language used for both system programming and application programming, making it often referred to as a Middle-Level Programming Language, supporting both low-level and high-level functionalities.

Features of C Language

  • Supports both high-level and low-level programming, making it a mid-level language.
  • Mostly used for system programming but also popular for application programming.
  • Program development is faster and more efficient.
  • Has powerful tools and keywords.
  • Uses its own text editor and compiler.
  • Supports structured programming.
  • Portable across platforms easily.
  • Offers a wide range of powerful library functions.
  • Flexible for different types of programming.

Advantages and Disadvantages of C Language

Advantages:

  • Suitable for system programming.
  • Easy to interact with hardware.
  • Easy to learn and understand.
  • Compact and efficient code.
  • Program code is secure.

Disadvantages:

  • No runtime checking.
  • No strict data type checking.
  • Debugging becomes complex with larger programs.
  • Requires semicolons for each statement.

Define the Following Terms:

Editor

The area where you type the program. C has its own text editor, but other text editors can also be used.

Pre-processor

A collection of complex codes used to instruct the machine about certain keywords. It is written with a preceding # symbol (e.g., #include).

Compiler

A software that converts the source code of high-level language into machine-readable object code.

Linker

A software that links library files with object codes and creates an executable file (with .exe extension) that can run independently.

Fundamentals of C

Character Set in C Language

The character set used in C includes:

  • Alphabets: A-Z, a-z
  • Digits: 0-9
  • Special Symbols: ~ ! @ % ^ & * ( ) - _ + = | \ { } [ ] : ; " ' <> , . ? /

Comments in C

Comments are non-executable text used for documentation. Two types of comments in C:

  • Single-line Comment: Uses // symbol (e.g., // This is my first program).
  • Multi-line Comment: Uses /* */ symbols (e.g., /* This is a multi-line comment */).

C Tokens

C Tokens are the smallest units that remain unchanged during compilation. They include identifiers, keywords, constants, variables, and operators.

Identifiers

Identifiers are names used for variables, functions, arrays, etc., in a C program.

Keyword

Keywords are reserved words in C that cannot be used as identifiers. Example: void, int, if, etc.

Data Types in C

C supports the following data types:

  • int: Integer type (2 bytes of memory).
  • float: Real numbers with decimal points (4 bytes of memory).
  • char: Single characters (1 byte of memory).
  • void: Represents null or no value.

Constants and Variables

Constants are fixed values that do not change during program execution. Variables are names that store data that can change.

Statements in C

A statement is a sequence of expressions that perform an operation. Statements can be simple or compound.

Escape Sequences

Escape sequences represent special characters in a string. Some common escape sequences:

New Line: \n
Tab: \t
Null Character: \0
Quotation Mark: \"
Backslash: \\

Operators in C

Operators are symbols used to perform operations on operands. Some types of operators include:

  • Arithmetic Operators: +, -, *, /
  • Relational Operators: ==, !=, >, <
  • Logical Operators: &&, ||
  • Assignment Operators: =

Library Functions

Library functions are pre-defined functions that can be used to perform common tasks. Example functions include printf(), scanf(), abs().

Input/Output Functions

Common I/O functions in C:

  • printf(): A formatted output function.
  • scanf(): A function to input values from the user.
