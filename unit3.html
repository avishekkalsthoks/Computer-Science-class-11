Operating System Types

A) On the basis of processing

1) Multi-programming OS

Multi-programming is a technique used in multi-user environments where multiple user programs are executed simultaneously by a single processor. It aims to increase the utilization of computer resources by executing portions of different programs alternately.

Types: Multi-tasking, Multi-processing, Multi-user.

2) Multi-tasking OS

Multi-tasking OS allows multiple programs to run concurrently, primarily in a single-user system. The system can execute tasks simultaneously, either on a single processor or multiple processors. Examples include MS-Windows, Linux, and MAC OS.

3) Multiprocessing OS

Multiprocessing systems have more than one processor linked together to run programs on multiple CPUs simultaneously. It supports parallel processing, which increases program execution speed. It is costlier and more complex than multi-programming. UNIX is a common example.

4) Time-Sharing OS

Time-sharing enables multiple users located at different terminals to access the computer system simultaneously by sharing CPU time. Processes are switched frequently so that users receive immediate responses.

Advantages: Quick response, reduces CPU idle time.

Disadvantages: Reliability issues, security concerns, integrity problems.

5) Batch Processing OS

Batch processing is used for jobs that do not require immediate interaction. Jobs are grouped into batches and processed one by one. It is often used for tasks like payroll, statistical analysis, etc. Users submit their programs and retrieve results later.

Disadvantages: Lack of interaction during processing.

6) Multi-threading OS

In multi-threading OS, a program is divided into smaller sub-processes called threads. These threads are executed concurrently, increasing efficiency and task execution speed.

7) Real-Time OS

Real-time OS processes data quickly to control environments within a predefined time. It is critical for tasks where timing is essential, such as in flight seat availability systems, rocket launching, and manufacturing processes.

8) Online Processing OS

Online processing allows transactions to be processed immediately as they happen. Users can interact with the system during processing, and the results are provided once processing is complete. It is simpler than real-time OS.

B) On the basis of User Interface (UI)

Here are the various types of user interfaces in operating systems:

  • Command Line Interface (CLI): Allows users to type commands to interact with the system.
  • Graphical User Interface (GUI): Provides visual interaction using icons and windows.
  • Touch User Interface (TUI): Optimized for touch- based interactions, found in smartphones and tablets.
  • Voice User Interface (VUI): Allows users to interact with the system using voice commands.
