Chapter 4: Application Packages

MS-Word (Word Processor)

Introduction: MS-Word is a text editing or word processing software developed by Microsoft Corporation as part of the MS-Office package. It enables users to create, edit, and print documents with powerful features for text manipulation, grammar checking, image insertion, and more.

File Extensions: ".doc" for older versions, ".docx" for newer versions.

Features of MS-Word

  • Fast Operation: MS-Word operates faster as it is a digital text editor with no mechanical movement.
  • Editing Text Features (Character Formatting): Customize documents with bold, italic, underline, different font sizes, and styles.
  • Permanent Storage: Files can be stored on a disk or other storage media for easy retrieval.
  • Graphics: Insert pictures and graphs into documents.
  • OLE (Object Linking and Embedding): Share data between different programs.
  • Mail Merge: Generate multiple letters, labels, envelopes, etc.
  • Searching: Search and replace text throughout the document.
  • Printing: Print documents anytime.
  • Formatting: Enhance document appearance with alignment, line spacing, borders, and more.
  • Spelling and Grammar Checker: Check spelling and grammar errors with color-coded underlines.
  • Thesaurus: Look up synonyms and antonyms for words.

MS-Excel (Spreadsheet)

Introduction: MS-Excel is a spreadsheet program part of the MS-Office suite. It allows users to store, organize, and analyze data using formulas, charts, and graphs. Files have the extensions ".xlsx" and ".xls".

Common Terminologies Used in Spreadsheet

  • Worksheet: A collection of rows and columns to organize and calculate data.
  • Workbook: A file that contains one or more worksheets.
  • Cell: The intersection of rows and columns in a worksheet (e.g., A1, B2).

Features of MS-Excel

  • Supports multiple worksheets.
  • Create charts and graphs for data presentation.
  • Organize and manage large datasets.
  • Perform calculations automatically using formulas.
  • Sort data in ascending or descending order.
  • Support for object linking and embedding.
  • Format data using font size, color, and more.
  • Simple computing operations like copy, paste, find, and replace.
  • Includes internet features like web toolbar.

MS-PowerPoint (Presentation)

Introduction: MS-PowerPoint is a presentation software in the Microsoft Office suite that allows users to create professional presentations using slides. Files have the extensions ".ppt" and ".pptx".

Features of MS-PowerPoint

  • Format data with fonts, colors, and hyperlinks to create interactive presentations.
  • Use pre-defined layouts, themes, and templates for faster development.
  • Create custom animations and slide transitions.
  • Support for internal and external hyperlinks.
  • Display data using various charts, graphs, and visuals.
  • Utilize web tools, online help, and web linking features.
